# Baby rhythm PWA

Static PWA version for iPhone and Cloudflare Pages.

## Cloudflare Pages

Use this folder as a standalone Pages project.

- Build command: leave empty
- Build output directory: `baby-pwa`
- Custom domain example: `baby.your-domain.com`

If this folder is copied into an existing site repository, keep the same settings and point the output directory to the copied folder.

## iPhone install

Open the deployed URL in Safari, tap Share, then Add to Home Screen.
